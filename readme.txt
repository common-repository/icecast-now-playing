Displays Icecast current song playing, number of listeners, and a link to play the audio stream.

== Description ==
Displays Icecast server connection statistics for a given mount point. Allows visitors to listen to your audio stream from your blog, view the current tracks playing, and number of listeners. To see the plugin in action visit http://www.galwayland.com/icecast-now-playing/.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How many mount points does the plugin support? =
The plugin supports 3 mount points.

= What version of Icecast does the plugin support? =
The plugin is tested up to Icecast version 4.9.4 .
